## Quick Start

### Local Development
```bash
pip install -r requirements.txt
python app.py
```
Visit: http://127.0.0.1:5000

### Deploy to Vercel
```bash
vercel
```

## Environment Variables
- `TELEGRAM_BOT_TOKEN` - Your Telegram bot token
- `TELEGRAM_CHAT_ID` - Your Telegram chat ID
